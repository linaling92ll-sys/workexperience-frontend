# Work Experience Frontend

## Beskrivning
Denna webbapplikation är en frontend som kommunicerar med en REST-baserad webbtjänst för att hantera arbetserfarenheter.

Användaren kan:
- Se alla arbetserfarenheter
- Lägga till nya poster
- Ta bort poster

Applikationen använder Fetch API för att kommunicera med backend.

---

## Tekniker
- HTML
- CSS
- JavaScript
- Fetch API

---

## Koppling till backend
Frontend skickar anrop till följande API:
http://localhost:3000/api/work
